# Nodejs Express Prisma API Channel 13 with Typescript

How to use this server:

## Setup

```
npm install
```

## Setup mysql and schema
```
.env.example to .env and configure the port and schema
```

## Setup prisma
```
npx prisma migrate dev --name init
```

## Seed Tasks
```
npx prisma db seed
```

## Development

```
npm run dev
```
